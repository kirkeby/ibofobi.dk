Jukebox

What is this?!

Jukebox is an MP3 streaming server with built-in web- and telnet-admin interfaces, and support for either acting as an encoder for an icecast server or running it's own streaming server.

Jukebox has a two-level playlist, with the highest level being the wishlist, and the lowest level being the randomized playlist. So, when you wish one or more songs they will play in order, and when the playlist is empty songs will be chosen at random from your MP3 directory.

Download

Jukebox requires Sheared, so you must install it first.

If you want to run the web-admin interface you will also need to install Entwine.

Then you will be ready to go download Jukebox.
